# Computer Engineering Fundamentals

This repository contains a comprehensive personal academic study covering the fundamental concepts of Computer Engineering. The document is designed to provide both theoretical understanding and system-level insight into how computers work.

---

## 📌 Overview

This study explores the core building blocks of computer systems, including both hardware and software aspects. It focuses not only on definitions but also on how systems operate internally.

---

## 🧠 Topics Covered

### 🔹 Computer Hardware
- Central Processing Unit (CPU)
- Memory (RAM, ROM, Storage)
- Input / Output Systems
- Computer Networks

### 🔹 Computer Software
- Software Fundamentals
- Operating Systems
- Number Systems
- Algorithms
- Data Structures
- Programming Languages
- Databases
- Data Security

---

## ⚙️ Key Concepts Explained

- Instruction cycle (Fetch – Decode – Execute)
- Memory hierarchy and paging
- Process scheduling and multitasking
- Algorithm design approaches:
  - Divide & Conquer
  - Iterative vs Recursive
- Time & Space Complexity (Big-O)
- Data organization in memory
- Database systems and SQL fundamentals
- Basic principles of data security
